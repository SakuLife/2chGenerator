## 📦 納品パッケージの準備

### 1. 納品ファイルの構成

顧客に渡すZIPファイルには以下を含めます：

```
2ch-video-generator.zip
├── 📁 2ch-video-generator/
│   ├── main.py
│   ├── setup.py
│   ├── create_sample_assets.py
│   ├── requirements.txt
│   ├── .env.example          ← 重要
│   ├── .gitignore
│   ├── README.md
│   ├── QUICKSTART.md
│   ├── INSTALLATION_GUIDE.md  ← 顧客向けマニュアル
│   └── src/
│       ├── __init__.py
│       ├── config.py
│       ├── 1_script_gen.py
│       ├── 2_image_gen.py
│       ├── 3_voice_gen.py
│       └── 4_video_edit.py
└── 📄 はじめにお読みください.txt
```

### 2. 除外すべきファイル

以下は**絶対に含めない**でください：

❌ `.env` （実際のAPIキー入り）
❌ `generated/` フォルダ（生成ファイル）
❌ `__pycache__/` （キャッシュ）
❌ `.git/` （Git管理情報）

### 3. 「はじめにお読みください.txt」の作成

```text
===================================================
  2ch/5ch まとめ動画 自動生成システム
===================================================

この度はご購入ありがとうございます！

【重要】最初にやること
1. INSTALLATION_GUIDE.md を開いてください
2. 手順に従ってセットアップしてください
3. わからないことがあればサポートにご連絡ください

【ファイル説明】
- INSTALLATION_GUIDE.md : 詳しいセットアップ手順
- QUICKSTART.md         : 5分でわかるクイックガイド
- README.md             : 機能詳細・カスタマイズ方法

【サポート】
メール: support@example.com
営業時間: 平日 10:00〜18:00

それでは、素敵な動画作成を！
===================================================
```

## 🛠️ サポート体制の構築

### よくある問い合わせと回答

#### Q: 「Pythonって何ですか？難しいですか？」

```
A: Pythonはプログラミング言語の一つですが、
このシステムではプログラミングの知識は不要です。

インストール方法はマニュアルに画像付きで
詳しく書いてありますので、指示通りに進めれば
どなたでも使えます。

もしつまずいた場合は、画面のスクリーンショットを
お送りいただければサポートいたします。
```

#### Q: 「OpenAI APIの料金が心配です」

```
A: OpenAI APIは従量課金制で、使った分だけ課金されます。

【料金目安】
- 1動画: 約¥350
- 月10本: 約¥3,500
- 月30本: 約¥10,000

YouTube動画を外注すると1本¥5,000〜¥30,000
かかることを考えると、非常にコストパフォーマンスが
良いと言えます。

また、最初は$5（約¥750）からお試しいただけます。
```

#### Q: 「Macでも動きますか？」

```
A: はい、Mac版の詳しい手順もマニュアルに
記載しております。

Windows・Mac両方に対応しており、
動作に差はございません。
```

#### Q: 「商用利用できますか？」

```
A: はい、YouTubeでの収益化も可能です。

ただし、OpenAIの利用規約を遵守する必要があります。
具体的には以下の通りです：

✅ OK: YouTube投稿・収益化
✅ OK: 自社メディアでの使用
❌ NG: 違法コンテンツ・誹謗中傷
❌ NG: スパム的な大量生成
```

| **Brain** | 12% | アフィリエイト可 | 競合多い |
| **Gumroad** | 10% | 海外顧客も | 英語必要 |
